## 技術的拡張ポイント

### すぐに追加可能（既存アーキテクチャで対応）

| 拡張 | 理由 |
|------|------|
| 新アクションタイプ | ActionType enum に追加 + BattleManager.ExecuteAction() に分岐追加 |
| 新バフタイプ | StaffBuffType enum に追加 + StaffManager.AccumulateBuffs() に分岐追加 |
| 新料理カテゴリ | DishCategory enum に追加 + MealBuffApplier に分岐追加 |
| 新属性 | ElementType enum に追加 + CharacterStats に耐性フィールド追加 |
| 新家具タイプ | FurnitureType enum に追加（ボーナス計算は変更不要）|
| 新カレンダーイベント | CalendarEventData アセット追加のみ |

### 設計変更が必要

| 拡張 | 変更箇所 |
|------|---------|
| マルチパーティ | BattleManager の PlayerParty 管理 + 編成UI |
| PvP | BattleManager を2プレイヤー対応 + ネットワーク層 |
| リアルタイム経営 | DinerService を時間経過シミュレーションに変更 |
| ガチャ/課金 | 完全新規システム |
